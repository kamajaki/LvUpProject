## 초보자를 위한 빠른 참고

### 초반(1~30) 추천 카드 포인트
- **포링/루나틱**: LUK·회피·크리 입문용
- **히드라**: 인간형(플레이어 포함) 사냥·PVP 핵심
- **타라 프로그**: 인간형 피격 감소 (탱크형)
- **스켈레톤**: 스턴 유틸
- **크리미**: 텔레포트 이동 편의

### 중반(40~70) 추천 카드 포인트
- **데니로/피에르/앙드레**: ATK + PEN (딜러 성장)
- **아르지오페**: 갑옷 독 속성화
- **무카**: HP 회복·마법 피격 감소
- **페코페코**: HP 탱킹
- **데저트 울프**: 소형 몬스터 특화

### 후반(70+) 추천 카드 포인트
- **심연의 기사**: Mini/MVP 특화
- **나인테일 / 와일드 로즈**: 회피형
- **하이 오크**: 데미지 반사
- **이블 드루이드**: 언데드 속성화 + 힐 보너스
- **레이드릭**: 무속성 공방

### 용어 메모
- `-` = 데이터 없음 / 미확인
- `[1]` = 슬롯 1개 있는 장비
- PEN = 관통(방어 무시 관련)
- Ignore DEF/MDEF = 방어/마법방어 무시
- Mini / MVP = 미니보스 / 보스
